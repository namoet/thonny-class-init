This Thonny IDE plugin creates a new option under the "tools" toolbar named "Fill class init!".
It can be used to copy the parameters of the __init__ function of a class to class members with the same name.

<h1>Installation</h1>
To install the plugin, paste the python file in a directory called "thonnycontrib" lying in a directory in the python path.
For example use:
%appdata%\Thonny\plugins\Python310\site-packages\ under windows.

<h2>Usage</h2>
When writing the __init__ function of a class place the cursor under the definiton line of the function press "tools"->"Fill class init!":<br>
